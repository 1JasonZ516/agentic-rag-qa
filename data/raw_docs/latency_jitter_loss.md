# Latency, Jitter, Loss (Definitions)
Latency is end-to-end delay. Jitter is variation in delay over time; high jitter harms VoIP because packets arrive unevenly. Loss is the fraction of packets that never arrive; loss reduces quality for real-time media and triggers retransmissions in TCP.
Monitoring often tracks p50/p95/p99 latency, jitter statistics, and loss rate.
